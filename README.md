# Blitz

> Typing test measuring speed and accuracy

## Running Locally

1. Clone the repository

```sh
git clone https://github.com/ldgni/blitz.git
```

2. Navigate inside the repository

```sh
cd blitz
```

3. Install dependencies

```sh
pnpm install
```

4.  Start the development server

```sh
pnpm dev
```
